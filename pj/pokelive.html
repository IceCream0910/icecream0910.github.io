Summary

포고인들을 위한 커뮤니티 레이드, 친구요청, 교환 등 오직 포고인들을 위해 구성된 커뮤니티는 포켓몬고의 플레이를 더 재밌게 만듭니다. 게임에서 진행되는 다양한 이벤트 소식을 세번째 탭에서 한번에 모아볼 수 있습니다. 게임 플레이 중 질문이 생긴다면 포켓라이브에서 질문해보세요.

어려웠던 점 & 해결방안

안드로이드에서 키패드가 표시되면 웹페이지 하단이 잘리는 문제가 발생했다.
안드로이드에 웹뷰를 사용할 때 상단바를 숨기면 키보드가 올라올 때 웹뷰의 크기가 조절되는 동작이 작동하지 않는다는 것을 발견하였다.

그누보드 smarteditor 적용 시 모바일 UI가 잘리는 문제
게시판 글 작성을 위한 에디터에 네이버의 smarteditor을 적용했는데, 모바일에서 편집 UI가 잘리는 문제가 발생했다. 모바일에서도 깨지지 않고, 동영상이나 이미지를 첨부하기에 더 편리한 summernote 에디터를 적용해 해결했다.

새롭게 알게된 점

PHP 게시판 개발
그누보드라는 게시판 제작 툴을 이용해 서비스를 개발해보았다. 서비스 기획에 맞게 레이아웃과 기능들을 수정하며 php 환경을 경험하게 되었다.

SQL DB 사용
게시판과 회원 기능을 구축하며 SQL 데이터베이스를 사용해보았다. SQL DB 관리에 사용되는 명령어들이나 기본적인 구조들을 다양한 레퍼런스들을 참고해 개발을 진행했다.
